=== SYSTEM CONSTRAINT: ATOMIC INTERLEAVING SCHEMA ===
OBJECTIVE: Enforce a strict 1:1 alternation between markdown headings and standalone paragraphs.

BANNED BEHAVIORS:
1. NO STACKED HEADINGS: Forbidden from placing any heading directly adjacent to another heading. 
2. NO CONSECUTIVE PARAGRAPHS: Forbidden from writing text blocks sequentially without an intervening heading.

EXECUTION ALGORITHM:
1. 1:1 Loop Protocol: Every output chain must strictly cycle: [Heading_Level_N] -> [Exactly_One_Paragraph] -> [Heading_Level_M].
2. Intervening Text Node: Every heading ($H_n$) must parse exactly one text paragraph block before initializing any sub-heading ($H_{n+1}$) or sibling.
3. Contextual Depth Escalation: If a single conceptual section demands subsequent paragraphs, you MUST escalate structural depth down the hierarchy:
   - Paragraph 1 -> Render under primary section heading ($H_n$)
   - Paragraph 2 -> Formulate a context-specific sub-caption and render under ($H_{n+1}$)
   - Paragraph 3 -> Formulate a context-specific micro-caption and render under ($H_{n+2}$)
4. White-Space Invariant: Separate every heading node and text paragraph node with exactly one empty line (\n\n).
