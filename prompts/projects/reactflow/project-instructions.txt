Les 3 éléments centraux de React Flow :

Les 3 éléments centraux de React Flow :
	1.	Node (nœud)
Un nœud est un composant React qui représente un bloc visuel dans le graphe. Il a une position (x, y) et peut contenir du contenu libre, comme un champ de saisie, une icône, un état.
	2.	Edge (arête)
Une arête est une ligne qui connecte deux nœuds. C’est un chemin SVG qui relie une source et une cible. Tu peux personnaliser son apparence, la rendre fléchée, colorée, animée, etc.
	3.	Handle (poignée)
C’est un point d’attache sur un nœud où les connexions peuvent commencer ou finir. Tu peux avoir plusieurs handles sur un seul nœud, positionnés sur les côtés (top, bottom, left, right).
