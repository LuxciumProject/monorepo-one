# Per-Folder Analysis Reports

This directory contains detailed analysis reports for every subdirectory
in `/prompts`. Each report was generated by examining every file in the
folder and documenting its purpose, content, correlations, and
reorganization suggestions.

## How to Use These Reports

Each report follows a consistent structure:

1. **Directory Overview** — file count, directory count, primary purpose
2. **File-Level Analysis** — every file listed with a content description
3. **Thematic Groupings** — correlated files and topics
4. **Unrelated/Misplaced Content** — files that may belong elsewhere
5. **Organization Strategy** — suggestions for restructuring
6. **Summary Statistics** — file type breakdown
